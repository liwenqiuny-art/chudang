GitHub 同步设置

题库编辑(改答案、补答案、加解析)只会先存在浏览器本地。在这里配置 GitHub 后,点"上传题库到 GitHub"可以把改动写入 questions.json,这样换设备也能看到。

需要 repo 权限。GitHub → Settings → Developer settings → Personal access tokens 创建。

本地数据管理

所有本地编辑、批注、错题、做题记录均存于浏览器。导出可备份,清空会全部删除。

部署提示
· 把 index.html、articles.json、questions.json 放在同一目录
· 上传到 GitHub 仓库根目录,在 Settings → Pages 启用 Pages
· 也可以在本地双击 index.html 打开使用(但 GitHub 同步只在网络访问时可用)
